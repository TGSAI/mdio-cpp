Detailed Description

template<typename T, DimensionIndex R, ArrayOriginKind OriginKind>
struct mdio::VariableData< T, R, OriginKind >

The in-memory representation of the data This object should only be constructed through the Variable::Read() function. This data only exists in memory and should be preserved with the Variable::Write() function.
