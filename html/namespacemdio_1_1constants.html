Detailed Description

A collection of frequently used constants when working with MDIO. Currently it contains the following frequently used constants:

  • OpenMode: Common operators for opening or creating a file.
  • dtypes: Supported dtypes specifically called out by the MDIO V1.0 specification. These dtypes are intended for Variable type-checking and not type-casting.
