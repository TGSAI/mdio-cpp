Detailed Description

A descriptor for slicing a Variable. A struct representing how to slice a Variable or Dataset. All slices using this will be performed as half open intervals.

Parameters
labelThe label of the dimension to slice.
startThe start index of the slice.
stopThe stop index of the slice.
stepThe step index of the slice.

Usage This provides an example of describing a slice for [0, 100) with a step of 1 along the inline dimension and [0, 200) with a step of 1 along the crossline dimension. The depth dimension would remain wholely intact as it is not specified.

mdio::SliceDescriptor desc1 = {"inline", 0, 100, 1};
mdio::SliceDescriptor desc2 = {"crossline", 0, 200, 1};
